# HexToRgb

Turn a hexadecimal color into its corresponding RGB, decimal values!

## Install

```
gem install hex_to_rgb
```

## Usage

```
> require 'hex_to_rgb'
> HexToRgb.new("#FFFFFF").rgb
=> [255, 255, 255]
```

## Development

### Install

```
gem install --dev hex_to_rgb
```


### Specs

The default Rake task is to run the specs.

```
rake
```
